Инструкция.
Администрирование сайта catsekb.ru

Для того, чтобы информация на сайте всегда была в актуальном состоянии существует панель администратора, проще говоря, "админка". 
Данная панель доступна по адресу: catsekb.ru/admin. Для доступа в панель вам потребуется учетная запись администратора.
Любое поле, которое выделено жирным в панели администратора является обязательным для заполнения.

После входа в панель администратора, перед вами появится несколько разделов:
    - КОТИКИ (управление животными)
    - ПОЛЬЗОВАТЕЛИ И ГРУППЫ (Управление аккаунтами админастроров)
    - СТАТЬИ (управление новостями и статьями)

Так же в верхней правой части имеются ссылки на сайт, сброса пароля и выхода из панели администратора.

	
КОТИКИ:
    
Раздел КОТИКИ:
    Раздел котики содержит 2 пункта: Группы, Питомцы.
    - Группы:
        На сайте предусмотрены 3 стандартные группы, принадлежность к одной из которых должен иметь каждый питомец. Это группы "Пристроен, Дома, На радуге".
        Название и описание данных групп из панели администратора изменить нельзя, но если появится такая необходимость, можно внести соответствующие изименения.
        Принадлежность к какой либо группе можно изменить у каждого животного в любое время. Эти группы отображаются в разделе меню "Наши коты". 
        Раскрытое меню "Наши коты" для пользователей (не админов) выглядит таким образом (фото 03). Для админов, которые зашли на сайт (фото 04). 
        Для того, чтобы не выходить из панели администратора, но посмотреть как будет выглядеть сайт для рядовых пользователей, можно зайти в режим инкогнито (Google Chrome - Ctrl+Shift+N)
        Как вы уже заметили, группа "На радуге" не отображается для обычных посетителей, по причине, что эта группа может вызвать грусть и уныние, вместо позитивных эмоций.
        
        Так же предусмотрены группы, которые может создавать, изменять и удалять админастратор. Такие группы находятся в разделе "Котики".
        Группы объединяют питомцев по какому либо принципу. К примеру можно создать группу "На лечении", добавить описание. Далее пре редактировании или создании питомцев, эта группа станет доступной для выбора.
        Группа содержит название, описание и галочку "Показывать группу".       
		Галочка "Показывать группу" скрывает данную группу для показа. Если группа имеется, она отоборажется в меню. 
        Скрытая группа всегда отображается для администраторов, которые осуществили вход на сайт в панели администратора. Проверить данную настройку можно используя режим инкогнито.
		Название группы отображается в разделе меню "Котики", а так же в качестве заголовка при переходе на данную группу.
        Описание отображается при переходе на страницу данной группы.
        
    - Питомцы:
        Для создания питомца достаточно нажать на кнопку "добавить питомец (может исправить в админке на "котика"?)" и нажать на кнопку сохранить. 
        При этом в базе данных котик получит собственный ID. Данный айди фигурирует в ссылке на странице этого питомца, а так же в ссылке для изменения на панели администратора.
        http://127.0.0.1:8000/cats/cat_256/
        http://127.0.0.1:8000/admin/cats/animal/256/change/
        Стоить отметить, что данный питомец не будет иметь никакой информации и абсолютно бесполезен. В панели администратора при редактировании питомца находится кнопка "Смотреть на сайте".
        При переходе по этой кнопке мы увидим пустую страницу с данным питомцем.
        Для добавления фото, необходимо привязать данного питомца к альбому в VK. Для этого необходимо открыть раздел "Импорт данных из группы в VK (Показать)", затем добавить ссылку на альбом ВК и нажать на кнопку "Сохранить и продолжить редактирование".
        ID альбома в ВК привязывается к данному питомцу.
        После того, как вы привязали альбом и сохранили, можно будет использовать кнопку "Обновить фото". Обновление фото происходит автоматически. - "неочень понял этот пункт (Киселев М.В.)"
        Так же можно использовать кнопку "Обновить описание и имя кота. :". Данная информация появится в соответствующих полях: "Имя" и "Описание". Для сохранения изменений требуется нажать на кнопку сохранения, но при этом не стоит забывать, что в ВК имеется ограничение на количество символов описания альбома.
        
        Стоит отметить что к 2 разным питомцам нельзя привязать 1 альбом. (Информация требует уточнения.)

        При необходимости, можно выставить дату появления в приюте. 
        (При автоматическом заполнении базы, это дата соответствовала дате созданию альбома.) 
        Это дата используется при отображении информации о времени нахождении в приюте, но только в том случае, если питомец имеет статус: "В приюте".

        Показывать котика - опция которая позволяет скрыть питомца от пользователей (но не от администратора(ов)). Например, если данный питомец не имеет фото. Для проверки используйте режим инкогнито.
        
        Проверено - эта опция, которая позволяет выделить тех питомцев, которые созданы вручную и имеют актуальную информацию. Это необходимо для постепеного перехода в актуальное состояние базы данных, так как первоночально база создалась в автоматическом режиме.
        
        Группа - позволяет выбрать принадлежность к группе, которая создается администратором вручную. Если группа скрыта, то животное не будет отображаться для рядовых посетителей (информацию нужно проверить).

        Пол - тут вопросов возникнуть не должно.

        Настройка тега, тег животного без "#" и "_catsekb".
        Поле текущий тег обновится только после сохранения. Нажмите на кнопку "сохранить и продолжить редактирование" и проверьте корректность данного тега перейдя по ссылке в данном поле. 
        (Для отображения всех фото по ссылке тега, желательно выполнить вход в инстаграме.)
        Во вкладке "Возраст котика" можно указать возраст (точный возраст указывать не обязательно).
        Во вкладке фотографии, можно посмотреть фото питомца.

        Для того, чтобы поместить фото на "аватарку", нужно поставить отметку - "Избранное", 
        "Использовать для фона страницы" - помещает фото в шапку страницы. Если изображение съехало вверх или вниз, установите значение "Позиция по вертикали". 
        Если фото не подходит, уберите эту опцию (при этом в шапке страницы будет использоваться стандартное (дефолтное) изображение).
        Совет: не используйте одинаковые изображения для аватарки и для шапки, это смотрится хуже.
	
		Шапка - верхняя часть страницы с фото.


РАЗДЕЛ СТАТЬИ:
    - НОВОСТИ:
        Автор - при необходимости к новостям и статьям можно добавиь автора. Это строка будет отоборажаться внизу новости или статьи.
        Новости. При добавлении новости необходимо указать заголовок.
        Текст новости следует писать вместе с html разметкой.
        Новости, в которых стоит отметка "скрыть новость", видны только при входе в панель администратора.
        "Важная новость" - выделяет новость в ленту. При этом новость занимает 2 колонки. Самая последняя важная новость отображается на стартовой странице.
        "Изображение для превью" - ссылка на изображение, которое отобразится на шапке страницы новости, и превью.
        "Позиция по вертикали" - для перемещения фото в шапке странице, по аналогии как с питомцами.
        "Автор" - при необходимости можно выбрать автора из списка.
		
		Желательно добавлять, например, по 2 обычные новости и затем важную (иначе обычная новость может оказаться одна в колонке).

    - СТАТЬИ:
        Со статьями дела обстоят практически так же, за исключением нескольких моментов.
        Для каждой статьи нужно добавить тему статьи. Темы добавляются в соответствующем пункте меню, раздела СТАТЬИ. Темы отображаются в меню.
        Есть стандартные статьи, названия которых нельзя изменять, но можно менять содержимое. Это статья "Что делать, я нашел кота?" , "Контакты", "О нас".
